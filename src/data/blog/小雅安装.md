# 基础文件

安装emby的媒体目录，一定要先把这几个文件提前下载好，不然过程非常慢all.mp4，115.mp4都是30+G大小

```
/opt/media/temp/all.mp4

/opt/media/temp/config.mp4

/opt/media/temp/115.mp4
```

安装小雅，emby的脚本

```
./xiaoya.sh
ls /opt/media/temp
mv /home/ftpuser/*.mp4 /opt/media/temp/
```

# emby全家桶大致安装步骤

1. 检查xiaoya目录
2. 检查磁盘空间185G以上
3. 下载all.mp4,config.mp4,114.mp4文件
4. 校验，解压以上三个文件
5. 安装emby容器
